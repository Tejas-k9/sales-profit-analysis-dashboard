# Sales & Profit Analysis Dashboard

## Problem Statement

The objective of this project is to analyze sales performance and identify the reasons behind low profitability despite strong revenue.

---

## Tools Used

* SQL (Data Analysis)
* Microsoft Power BI (Data Visualization)
* Microsoft Excel (Data Cleaning)

---

## Dataset

The dataset contains information on:

* Sales
* Profit
* Products
* Customers
* Regions
* Discounts

---

## Steps Performed

### 1. Data Cleaning

* Cleaned raw data using Excel
* Handled missing values and formatting issues

### 2. SQL Analysis

* Calculated total sales and profit
* Identified top and loss-making products
* Analyzed category and regional performance
* Evaluated discount impact on profitability

### 3. Dashboard Development (Power BI)

* Created KPI cards (Sales, Profit, Profit Margin, Loss %)
* Built category and region analysis visuals
* Identified loss-making products
* Analyzed customer profitability

---

## Key Insights

* 41% of products are loss-making
* Technology category contributes the highest losses
* High discounts negatively impact profitability
* Some high-sales customers generate negative profit
* Regional performance varies significantly
